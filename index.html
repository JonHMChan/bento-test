fundamentals

What to learn next

css

CSS is one of three major technologies you need to be a basic front end developer, giving you control over colors, positioning, and overall styles. You should know HTML before diving in.

html

HTML is one of the three major technologies you need to be a basic front end developer, giving you the essential tools to structure text, images, and links. This is where most beginners start.

javascript

JavaScript is one of three major technologies you need to be a basic front end developer, giving you the ability to add basic interactivity to your site. I recommend starting with HTML and CSS first though.
